Locate by Address

This method of location searching is best for users looking for point extreme precipitation data for a specific street address, city/province, postal code. Enter any address in the “Locate by Address” search box and either press Enter or click the right arrow button to search and select the best matching point.

Search Interface

The project makes use of the Google Maps API to geocode any well-formed address to usable latitude/longitude coordinates. Any combination of street address, city, province, postal code, business name, etc. can be entered in the search box (each element should be separated by a comma).

Sample address searches:

  • 1841 Argyle St, Halifax, NS B3J 3A5, Canada
  • 1841 Argyle St, Halifax, NS
  • 1841 Argyle St, NS B3J 3A5
  • Halifax, NS
  • NS
  • Novia Scotia
  • B3J 3A5

This system does have its flaws, as an address search may return multiple or incorrect locations. If you experience trouble finding a given address, it is recommended that use the regular Google Maps web site to locate it. You can then determine a more specific address, or double-click the Google map on the extreme precipitation web site to select the desired point.

Data are only available for the four Atlantic Canada provinces – New Brunswick, Nova Scotia, Prince Edward Island and Newfoundland (station, but not gridded data from Labrador is also included) and adjacent portions of the state of Maine. Selecting a point elsewhere will result in an error message or obviously odd results. In addition, the resolution of the project dataset is approximately 5 km, so a selected point will be matched with the closest grid point to pull the output data.
