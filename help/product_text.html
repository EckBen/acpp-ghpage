Extreme Precipitation Tables - Text/CSV


The Extreme Precipitation Tables product presents an easy way to extract data from the project database for a given point or station. Output will include return period estimates and confidence limits (upper and lower bounds) for selected durations and all recurrences.

The Text/CSV product produces comma delimited output that can be copied from the browser window and pasted into spreadsheet. This output is not designed for easy human viewing, but rather to import data into compatible software for analysis.
