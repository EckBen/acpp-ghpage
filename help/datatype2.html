Data Type

Used only on the GIS data files product, this dialog allows users to display gridded files containing the Return Period Estimates (the "values"), or either of the confidence limits (Upper or Lower).

Unless you have a specific need to using the upper or lower bounds, it is highly recommended that the Return Period Estimates be used for design and research work.

Options:

  • Return Period Estimates
  • Upper Confidence Limits
  • Lower Confidence Limits

Products:

  • GIS Data Files
