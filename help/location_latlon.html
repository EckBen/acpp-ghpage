Locate by Lat/Lon

To search for a specific location using the latitude and longitude coordinates, enter them in the input boxes and either press Enter or click the right arrow button to select the desired point.

Search Interface

This search method only allows coordinates in decimal format. To convert from degree-minute-second (DMS) format, use the handy tool at https://www.fcc.gov/media/radio/dms-decimal

Negative signs are ignored, as these fields assume North (positive) latitude and West (negative) longitude. Data are only available for the four Atlantic Canada provinces – New Brunswick, Nova Scotia, Prince Edward Island and Newfoundland (station, but not gridded data, from Labrador is also included) and adjacent portions of the state of Maine. Selecting a point elsewhere will result in an error message or obviously odd results. In addition, the resolution of the project dataset is approximately 5 km, so a selected point will be matched with the closest grid point to pull the output data.
