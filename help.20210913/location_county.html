Locate by State/County

For users that only have a vague location, we have built-in a tool to automatically select the approximate geographic center of each province and, where appropriate, county in the project region. Please note, the point selected using this method is not necessarily a representative or average point for the entire province or county – it is simply the geocoded geographic center according to the Google Maps API.

To utilize this search method, select the desired province or county in the dropdown box and the center will automatically be selected.

Search Interface

Data are only available for the four Atlantic Canada provinces – New Brunswick, Nova Scotia, Prince Edward Island and Newfoundland (station, but not gridded data from Labrador is also included) and adjacent portions of the state of Maine. The resolution of the project dataset is approximately 5 km, so a selected point will be matched with the closest grid point to pull the output data.
