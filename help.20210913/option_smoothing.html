Smoothing

This option dictates whether the smoothing routine developed for the NRCS WinTR20 software package should be applied to data before output. Note that choosing No will NOT remove all smoothing from the analysis, producing "raw" results. The regionalization, gridding, and smoothing techniques employed in this project will always be present in data output. This extra smoothing step resolves some of the inconsistencies among the n-minute and hourly durations.

This smoothing step only affects the 5min, 10min, 15min, 30min, 120min, 3hr, 6hr, and 12hr durations. While the 60min and 24hr durations are used in the smoothing calculations, they remain unchanged.

For more information, please refer to the Technical Manual.

Options:

  • Yes
  • No
