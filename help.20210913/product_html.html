Extreme Precipitation Tables - HTML


The Extreme Precipitation Tables product presents an easy way to extract data from the project database for a given point or station. Output will include all return period estimates and confidence limits (upper and lower bounds) for all durations and recurrences.

The HTML product produces a series of easy-to-read tables compatible with any web browser. In addition, a Download button provides a means by which the data tables can be saved locally for future reference (default file name is extreme_precip_table.pdf).
