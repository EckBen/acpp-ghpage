Select Specific Stations

Checking the Select Specific Stations box changes the map selection interface. Red dots on the map, show the location of the hourly precipitation stations used in this and the EC IDF analysis.

Specific Station Selection Interface

From this interface, the other location selection options (including double-clicking on the map) are disabled. However, clicking on a station dot selects that specific station. The name of the selected station will appear in the Selected Station box on the interface. Once a station has been selected in this fashion, clicking the submit button will generate the desired extreme rainfall product using observed station data rather than values from the interpolated grid.

Selecting a different location, either another station or a point from the grid, requires the user to uncheck Select Specific Stations box. The original interface's functionality will be restored. If data from a different station is desired, the Select Specific Stations box must be re-checked.
