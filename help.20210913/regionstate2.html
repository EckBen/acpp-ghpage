Region/State

Used only for the Maps product, this location selector allows the user to select any of the state in the product or the entire region (Northeast US) to be displayed in a pre-processed map form. Please note that some maps may have a "buffer" built in, where the contour colors bleed slightly beyond the state or regional borders - this is normal.

Options:

  • Northeast US
  • Connecticut
  • Maine
  • Massachusetts
  • New Hampshire
  • New York
  • Rhode Island
  • Vermont
